### 📊 凯利公式计算说明

凯利公式: **f = (bp - q) / b**

其中:
- **b** = 赔率 = 平均盈利 / 平均亏损
- **p** = 胜率
- **q** = 败率 = 1 - p
- **f** = 建议仓位比例
